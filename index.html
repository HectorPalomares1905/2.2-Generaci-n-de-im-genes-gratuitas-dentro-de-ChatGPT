1. ¿Qué es DALL·E?

2 / 9
Definición
DALL·E es un sistema de inteligencia artificial desarrollado por OpenAI que genera imágenes a partir de descripciones escritas en lenguaje natural. Está integrado directamente en ChatGPT.
¿Cómo acceder gratis?
  • Entra a chat.openai.com
  • Inicia sesión con tu cuenta gratuita
  • En la conversación, escribe un prompt describiendo la imagen que quieres crear
  • ChatGPT usará DALL·E automáticamente para generarla
Plataforma¿Incluye generación de imágenes?
ChatGPT (versión gratuita)Sí, con DALL·E (cantidad limitada)
Gemini (gratuito)Sí, con Imagen de Google
Claude (gratuito)No genera imágenes
Dato clave
No necesitas saber de diseño ni de arte. Solo escribe lo que imaginas y DALL·E lo crea por ti.

2. Cómo crear imágenes con DALL·E

3 / 9

Pasos básicos

  1. Describe tu imagen: Escribe qué quieres ver (objetos, colores, estilo, ambiente).
  2. Agrega detalles: Mientras más específico seas, mejor será el resultado.
  3. Especifica el estilo: Realista, animado, acuarela, dibujo, 3D, etc.
  4. Itera: Si no te gusta, pídele ajustes a ChatGPT.

Consejos para un buen prompt visual

ElementoEjemplo
Sujeto principal"un gato", "una casa moderna", "un logotipo"
Entorno o fondo"en una playa", "en un bosque", "fondo blanco"
Estilo artístico"estilo acuarela", "fotorrealista", "animado", "dibujo a lápiz"
Iluminación"luz natural", "atardecer", "iluminación de estudio"
Colores"tonos pastel", "blanco y negro", "colores vibrantes"
Formato"horizontal 16:9", "cuadrado", "vertical"
Fórmula recomendada
"Crea una imagen de [sujeto] en [entorno], estilo [estilo], con [colores/iluminación], formato [horizontal/cuadrado/vertical]."

3. Ejemplos prácticos (1-3)

4 / 9

Ejemplo 1: Paisaje natural

"Crea una imagen de una montaña nevada con un lago cristalino al amanecer, estilo fotorrealista, colores fríos, formato horizontal."
Ideal para fondos de pantalla o presentaciones sobre naturaleza.

Ejemplo 2: Logotipo institucional

"Diseña un logotipo moderno para un instituto de capacitación, con un libro abierto y una estrella, colores azul y naranja, estilo minimalista, fondo blanco."
Útil para proyectos visuales sin saber diseño gráfico.

Ejemplo 3: Escena educativa animada

"Ilustración animada de un salón de clases con estudiantes diversos usando tablets, colores cálidos, estilo cartoon, formato cuadrado."
Perfecto para materiales educativos o presentaciones.

3. Ejemplos prácticos (4-6)

5 / 9

Ejemplo 4: Imagen para redes sociales

"Crea una imagen para Instagram de una taza de café humeante con hojas de otoño alrededor, estilo acuarela, tonos naranja y marrón, formato cuadrado."
Ideal para publicaciones de redes sociales sin usar bancos de imágenes.

Ejemplo 5: Producto ficticio

"Genera la imagen de un envase de perfume elegante de vidrio azul con un tapón dorado, sobre una superficie de mármol, iluminación de estudio, estilo fotorrealista."
Sirve para catálogos conceptuales o lluvia de ideas de diseño.

Ejemplo 6: Mascota / personaje

"Dibuja una abeja robot simpática con casco y herramientas, estilo animado 3D, colores amarillo y azul, fondo degradado."
Útil para crear personajes para campañas o juegos educativos.

Comparativa: prompt básico vs. detallado

6 / 9
TipoPromptResultado esperado
Básico "Un perro" Imagen genérica, puede no tener fondo ni estilo definido.
Detallado "Un perro golden retriever jugando en un parque soleado, estilo fotorrealista, colores cálidos, formato horizontal." Imagen específica, lista para usar en presentaciones o redes.
Básico "Una casa" Resultado simple, sin detalles arquitectónicos.
Detallado "Casa moderna de dos pisos con jardín frontal y piscina, estilo render arquitectónico, iluminación de atardecer, formato horizontal." Imagen profesional utilizable en presentaciones de proyectos.
Regla práctica
Tu prompt debe tener al menos: sujeto + entorno + estilo + iluminación/colores + formato.

4. Buenas prácticas

7 / 9
  • Sé descriptivo: No digas solo "un árbol", di "un árbol de cerezo en flor junto a un río".
  • Define el estilo: Especifica si quieres fotorrealista, dibujo animado, acuarela, 3D, etc.
  • Usa referencias de color: "tonos pastel", "colores vibrantes", "blanco y negro".
  • Ajusta el formato: "horizontal 16:9" para presentaciones, "cuadrado" para redes sociales.
  • Itera sin miedo: Pide cambios específicos si el resultado no es lo que esperabas.
Importante
DALL·E tiene un límite de generaciones gratuitas por día. Una vez alcanzado, puedes seguir al día siguiente o usar otras herramientas gratuitas como Bing Image Creator (también usa DALL·E) o Gemini de Google.
